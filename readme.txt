Adds custom tweet button to BuddyPress sites using YOURLS.

== Description ==

This version requires Buddypress and YOURLS url shortener.
Add tweet buttons to activity stream posts allowing users to 'tweet' the item. Shorten urls with your own custom YOURLS shortener. Other Url shortening services being added soon.

== Installation ==

= Automatic Installation =

1. From inside your WordPress administration panel, visit 'Plugins -> Add New'
2. Search for `BuddyPress Tweet Urls` and find this plugin in the results
3. Click 'Install'
4. Once installed, activate via the 'Plugins -> Installed' page

= Manual Installation =

1. Upload `bp-tweet-urls` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
